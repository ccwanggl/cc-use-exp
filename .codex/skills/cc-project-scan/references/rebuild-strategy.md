# Rebuild Strategy

已有根目录 `AGENTS.md` 时，按以下顺序判断：

1. 已包含 `<!-- AUTO:* -->` 区块
   - 默认执行增量更新
   - 只更新自动区块，不改用户手写区

2. 不包含自动区块，但结构清晰且用户手写内容较多
   - 推荐“追加自动区块”
   - 将自动区块追加到文件末尾，或用户指定位置

3. 不包含自动区块，且内容明显是旧模板、占位符未替换或结构混乱
   - 推荐“首次全量重建”
   - 必须先得到用户确认

4. 用户明确拒绝更新
   - 跳过写入
   - 明确提示：Codex 新 session 可能仍缺少项目级上下文

首次全量重建时要求：

- 先复用 `cc-project-init/assets/AGENTS.project.md`
- 将扫描结果填入自动区块
- 尽可能迁移旧文件中仍有价值的项目约定
- 不保留明显错误、空占位符或与当前仓库冲突的旧内容
